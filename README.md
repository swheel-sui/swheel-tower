# swheel tower game

[Swheel tower game](https://swheel.xyz/game/tower) package in Sui. Includes:

- **difficulty**: *Difficulty* object
- **game**: *Game* object
- **state**: *GameState* object
- **sui_game**: *Game* use SUI coin
- **swheel_game**: *Game* use SWHEEL coin
- **tower**: entry functions
